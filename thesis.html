研究背景与挑战

为什么选择这个研究课题？

研究背景

神经性厌食症（Anorexia Nervosa）是一种严重的精神疾病，发病率在全球范围内持续上升。早期症状难以识别，通常被忽视或隐藏。社交媒体（如Reddit）成为发现早期语言信号的重要来源。本研究旨在通过深度学习技术，从社交媒体文本中自动检测早期神经性厌食症风险。

研究挑战

⚠️
传统诊断局限：依赖临床访谈或问卷，主观性强、耗时长，难以大规模筛查
💻
计算效率问题：现有深度学习方法处理长文本时计算资源消耗大，复杂度高达O(n²)
🎭
情感信息利用不足：情绪分析通常作为独立任务，未能充分利用用户帖子中的情感线索

核心技术方案

创新性地融合多种前沿AI技术

🔤

XLM-RoBERTa

作为基础编码器，生成高质量的上下文语义表示，支持多语言文本处理，捕获用户帖子的深层语义特征。

🌊

Mamba

状态空间模型（SSM），用于捕获长距离依赖关系。相比Transformer，具有更低的计算复杂度，更适合处理长文本。

💭

Qwen3-Reranker

本地部署的0.6B参数模型，生成1024维情感嵌入，提取用户的情感特征（正面/中性/负面），为预测提供辅助信号。

🎯

多任务学习

创新性地将厌食症风险预测作为主任务，情感分类作为辅助任务，实现任务间的知识共享与迁移学习。

⚖️

Focal Loss

处理严重的类别不平衡问题（6.8:1），通过动态调整损失函数，使模型更关注难以分类的少数类样本。

🔄

情感特征注入

将Qwen3提取的情感嵌入与主模型输出进行特征融合，让模型理解用户帖子背后的情感状态。

from lxml import etree
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords
from nltk.stem import WordNetLemmatizer

def process_text(text, stop_words, lemmatizer):
    # 小写转换
    text = text.lower()
    # 分词
    tokens = word_tokenize(text)
    # 移除停用词 & 词形还原
    tokens = [lemmatizer.lemmatize(w) for w in tokens
              if w not in stop_words and w.isalpha()]
    return ' '.join(tokens)

🔧 模型训练 (PyTorch)

# XLM-RoBERTa + Mamba 多任务学习
from transformers import AutoModel, AutoTokenizer
import torch.nn as nn

class MultiTaskModel(nn.Module):
    def __init__(self, bert_path, n_class=2):
        super().__init__()
        self.roberta = AutoModel.from_pretrained(bert_path)
        self.mamba = MambaLayer(d_model=768)
        self.classifier = nn.Linear(768, n_class)
        self.focal_loss = FocalLoss(alpha=0.25, gamma=2.0)

    def forward(self, input_ids, attention_mask, qwen_feat):
        # RoBERTa 编码
        roberta_out = self.roberta(input_ids, attention_mask)
        # Mamba 长序列建模
        mamba_out = self.mamba(roberta_out.last_hidden_state)
        # 融合 Qwen3 情感特征
        fused = mamba_out + qwen_feat
        # 多任务输出
        return self.classifier(fused)

📊 模型评估

# 评估指标计算
from sklearn.metrics import precision_recall_fscore_support

def evaluate(model, test_loader):
    model.eval()
    all_preds, all_labels = [], []

    with torch.no_grad():
        for batch in test_loader:
            outputs = model(**batch)
            preds = torch.argmax(outputs, dim=1)
            all_preds.extend(preds.cpu().numpy())
            all_labels.extend(batch['labels'].numpy())

    # 计算各项指标
    precision, recall, f1, _ = precision_recall_fscore_support(
        all_labels, all_preds, average='weighted'
    )
    return {'precision': precision, 'recall': recall, 'f1': f1}
